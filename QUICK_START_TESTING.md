## Troubleshooting

### "The app looks the same!"
- Did you restart it? Close completely and reopen
- Check the window title - does it say "v1.0"?
- Try running from PyCharm Terminal, not Windows CMD

### "Fonts look weird"
- Segoe UI should be on all Windows 10/11 systems
- If not available, it falls back to system default
- This is expected behavior

### "My icon doesn't show"
- Did you place it in `assets/icon.ico`?
- Check the filename is exactly `icon.ico` (not icon.ico.ico)
- Restart the app after adding the icon
- Right-click the file, check size (should be a few KB)

### "Window is too small/big"
- Resize it! The app remembers window position
- Minimum size is 1000x700 (can't go smaller)
- Default is 1200x800

### "Something broke!"
- Note what you were doing when it broke
- Check the console for error messages
- Save your notes
- Report in next session
