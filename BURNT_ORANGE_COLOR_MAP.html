🔥 Orange Usage Philosophy

Use ORANGE for:

  • ✅ Active/selected states (tabs, buttons, rows)
  • ✅ Primary actions (Save, Generate Invoice, etc.)
  • ✅ Table/list headers
  • ✅ Focus borders on inputs
  • ✅ Hover states on interactive elements

Keep TAUPE for:

  • 📋 Main background (warm, not stark white)
  • 📋 Inactive tabs
  • 📋 Secondary surfaces
