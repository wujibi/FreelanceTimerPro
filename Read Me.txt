Steps:
Create a new folder wherever you want (e.g., time_tracker_app)
Save all 5 Python files in that folder
Open command prompt/terminal in that folder
Run: pip install reportlab
Run: python main.py
Pro tip:
The app will create a time_tracker.db file (your database) in the same folder where you run it, so choose a location where you're okay with that file living permanently.

The beauty of this app is that it's completely portable - you can even copy the entire folder to another computer and it will work as long as Python and reportlab are installed!
