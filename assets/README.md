# Assets Directory

## Icon File

Place your custom icon here as `icon.ico`

### How to Create an Icon:

#### Option 1: Online Tool (Easiest)
1. Go to: https://www.favicon-generator.org/
2. Upload a square image (PNG, JPG) - ideally 256x256px
3. Download the .ico file
4. Rename to `icon.ico` and place in this folder
5. Restart the app to see your custom icon

#### Option 2: From Existing Image
1. Use an online converter: https://convertio.co/png-ico/
2. Upload your logo/image (Latrat Logo V1.jpg would work!)
3. Download as .ico
4. Save as `icon.ico` in this folder
5. Restart the app



### Icon Recommendations:
- **Size:** 256x256px (will auto-scale to different sizes)
- **Format:** .ico (Windows icon format)
- **Style:** Simple, recognizable at small sizes (16x16, 32x32)
- **Colors:** Match your brand (your logo would be perfect!)
- **Background:** Transparent is best
